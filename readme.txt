See https://kivy.org/doc/stable/gettingstarted/installation.html#install-pip 
1. install virtual environment, pymongo, and kivy using pip
2. create kivy environment called kivy_venv
3. before using code, go to the folder containing kivy_venv
4. run "kivy_venv\Scripts\activate" (you may first have to run command line as Administrator)
5. now you are able to use the kivy virtual environment

To view MongoDB database:
1. Install MongoDB (Do "Complete" installation)
2. Open Mongo Compass
3. connect to "mongodb://localhost:27017"
4. now you can view what's in the database
